## 📌 Overview
This project uses machine learning and historical Walmart sales data to predict future weekly sales. It includes data preprocessing, visualization, and predictive analytics using Linear Regression in Python.

---

## 🚀 Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## ✨ Features
- Data preprocessing
- Sales prediction
- Data visualization
- Model evaluation
- Historical data analysis

---

## 📂 Dataset
Walmart historical sales dataset containing:
- Weekly Sales
- Temperature
- Fuel Price
- CPI
- Unemployment
- Holiday Flag

---

## 🧠 Machine Learning Algorithm
- Linear Regression

---

## 📊 Output
The project predicts future weekly sales and visualizes:
- Actual Sales
- Predicted Sales
- Accuracy of the model

---
